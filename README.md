### xm-rn-cli

这是一个通过 cli 一键化生成可快速开发的 react-native 的业务模板脚手架。
[github 地址](https://github.com/WTxiaomage/xm-rn-cli.git)

使用命令

```
yarn add xm-rn-cli -g
rn create myProject
```
